## Adding a semen collection

Semen collections are recorded on the **sire's** (male) animal profile.

1. Open the sire's animal profile.
2. Click the **AI / Semen** tab.
3. Click **Add Collection**.
4. Enter:
   - Collection date
   - Laboratory name and test ID (optional)
   - Sample type (fresh, chilled, frozen)
   - Number of straws
   - Quality metrics — motility, morphology (optional)
5. Click **Save**.

## Assigning to a tank

Each collection can be assigned to a storage tank and goblet:

1. Open the collection record.
2. Click **Assign to Tank**.
3. Select an existing tank, or click **Add Tank** to create one.
4. Select or create a goblet within the tank.
5. Click **Save**.

## Recording AI usage

When straws are used for an insemination, usage is recorded via the **Breeding** tab of the dam (female). Select **From Inventory** as the semen source and choose the straw lot — the straw count decrements automatically when the breeding record is saved.

You can view the full usage history for any sire by opening their **AI / Semen** tab and scrolling to **Used Inventory**.
