title: Backup, Restore & Data Export
description: Exporting data and restoring from backups as a system administrator.
---

## Exporting all data

CapraGene includes a full data export function:

1. In the admin panel, go to **System → Backup & Export**.
2. Click **Export All Data**.
3. The system generates a ZIP archive containing:
   - All animal records (CSV)
   - All registration data (CSV)
   - All user data (CSV)
   - All uploaded files (photos, documents)
   - A restore script
4. Download the archive when the export is complete.

## Restoring from a backup

To restore data from a previously exported archive:

1. In the admin panel, go to **System → Restore**.
2. Upload the backup ZIP file.
3. Follow the on-screen prompts.
4. The system will restore all data from the archive.

<Warning>
  Restore operations overwrite existing data. Only perform a restore on a fresh installation or when directed by a CapraGene administrator.
</Warning>

## Exporting registration certificates

Individual registration certificates can be printed or saved as PDF:

1. Open the animal's profile.
2. Go to the **Registrations** tab.
3. Find the approved registration and click **View Certificate**.
4. Use your browser's print function (Ctrl/Cmd + P) to save as PDF.
